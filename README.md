# Пульт охраны банка

Это внутренний репозиторий для сотрудников банка "Сияние". Если вы попали в
этот репозиторий случайно,то вы не сможете его запустить, т.к. у вас 
нет доступа к БД, но можете свободно использовать код вёрстки или посмотреть, 
как реализованы запросы к БД.

### Как установить

1. Склонируйте репозиторий:
   ```bash
   git clone [ваш-repo-адрес]
   ```
2. Python3 должен быть уже установлен. 
3. Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
   ```bash
   pip install -r requirements.txt
   ```
4. Сайт не запустится без установки дополнительных библиотек: `django` и `psycopg2`.
   ```bash
   pip install django
   ```
   ```bash
   pip install psycopg2
   ```
5. Создайте .env файл (запросите актуальные данные у администратора):
    ```ini
    ENGINE=django.db.backends.postgresql
    HOST=ваш_хост
    PORT=ваш_порт
    NAME=имя_бд
    USER=ваш_логин
    PASSWORD=ваш_пароль
    SECRET_KEY=ваш_секретный_ключ
    ```
### Запуск
   ```bash
   python main.py
   ```
